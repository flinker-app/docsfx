# IFC Viewer for Teams

![IFC Viewer for Teams](/_media/load-multiple-ifc-files-in-teams-ifc-viewer.png)

The Teams IFC Viewer brings IFC and BCF coordination directly into Microsoft Teams. Load models from SharePoint or Teams, review BCF topics in the same channel where your team already works, and keep all project data inside Microsoft 365.

| Add custom workflows to your data | ✓ | ✗ |
| AI-ready data architecture | ✓ | ✗ |

## Teams-specific coordination features

- Add a shared IFC Viewer tab to a team, channel, or chat.
- Load one model, multiple IFC files, or a whole folder of IFC and BCF files.
- Open BCF clash topics with their saved view context directly in Teams.
- Share the currently selected BCF topic as a Teams deep link.
- Configure a tab to always open on the same BCF issue by setting a default topic GUID.
