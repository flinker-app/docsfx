# Introduction

Flinker documentation helps BIM, project, IT, and development teams evaluate, deploy, and operate openBIM workflows in Microsoft 365.

Use these docs to understand how Flinker solutions work with IFC models, BCF issues, IDS requirements, SharePoint, Microsoft Teams, Power BI, Excel, and the IFC Viewer SDK.

## What you can find here

- **Product guidance** for IFC Viewer solutions in SharePoint, Teams, Power BI, Excel, and related Microsoft 365 workflows.
- **Security and data protection documentation** for IT, compliance, and procurement reviews.
- **Admin deployment guides** that explain approval paths, tenant configuration, and rollout responsibilities.
- **openBIM standards guidance** for IFC, BCF, IDS, georeferencing, and model data workflows.
- **Developer resources** for embedding IFC, BCF, and IDS workflows with the IFC Viewer SDK.
- **Release notes** for tracking product updates across deployment channels.

## Enterprise evaluation path

For a security or procurement review, start with:

1. [Architecture and data protection](ifc-viewer-architecture-and-data-protection.md)
